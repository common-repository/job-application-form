== Description ==

This plugin provide functionality to take job apllication enteries from frontend and displays it in backend with proper format. to display the form in frontend use shorcode "**[job-shortcode]**".

A few notes about the sections above:

*   "[job-shortcode]" is to display form in frontend
*   Designation will only be displayed is it added in back end
*   "Requires at least" is the lowest version that the plugin will work on
*   Tables are auto mantained




== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `job-application-form.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Its ready to use

== Frequently Asked Questions ==

= Is data of applicants matained in back end?  =

Yes it is.

= Data is displayed in Descending order? =

Yes but you can click and reorder it.
